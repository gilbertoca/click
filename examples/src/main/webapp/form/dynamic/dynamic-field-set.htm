This example demonstrates how to dynamically add fieldsets to a form.
  • The form is submitted using the JavaScript statement, form.submit(), which is set as the Checkbox onclick handler. Clicking the checkbox will submit the form. On the server, Form validation is switched off through, Form.setValidate(false).
  • The utility method ClickUtils.bind(Form) is used to bind and inspect the checkbox value before the page onProcess event.
  • If the checkbox is checked, the address fieldset is added to the form
